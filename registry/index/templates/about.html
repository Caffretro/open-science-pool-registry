About the HT Phenotyping Project

The HT Phenotyping Transfer Service

Data transfers from data sources to storage and processing servers are facilitated by HTCondor. The HT Phenotyping project uses HTCondor to transfer data in an automated, reliable, and secure manner.

Requirements for using the Transfer Service

To use the transfer service, you must have a properly set-up computer to run the "client":

  1. The computer must be running of our support platforms.
  2. You must have administrative privileges on the computer. For Ubuntu, this means you can run the sudo command.
  3. The computer should be powered on overnight (synchronization is scheduled to occur nightly, usually around midnight).
  4. The computer should have a reliable, high-speed internet connection.
  5. The storage drive holding the data to be synchronized should be connected to the computer at all times, and should always be located at the same place in the filesystem.
The transfer service is resilient to short power or internet outages, but you should take steps to improve availability where possible (for example, by powering the computer from an uninterruptible power supply).
